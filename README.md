# ykit-starter-vue
Ykit-Vue.js 示例工程

## Getting Started
```
$ git clone -b ykit-v1 https://github.com/roscoe054/ykit-starter-vue.git
$ cd ykit-starter-vue
$ npm install 或者 yarn
```

## Run Dev
```
$ npm start
```
访问 http://127.0.0.1:3000/index.html

## Build
```
$ ykit pack -m
```

访问 https://ykit.ymfe.org 来获取更多文档
