# ykit-starter-react
ykit-Vue.js 示例工程。

## Getting Started
```
$ git clone https://github.com/roscoe054/ykit-starter-vue.git
$ cd ykit-starter-vue
$ npm install 或者 yarn install
```

## Run Dev
```
$ npm run dev
```
访问 http://127.0.0.1:8080/ykit-starter-vue/index.html

## Build
```
$ ykit pack -m
```

## 关于 ykit
访问 https://github.com/YMFE/ykit 来获取更多文档。
